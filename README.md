# 水科学来水预报

## 项目介绍

1、本项目采用Vue2 + Element UI + Vuex + Echarts + three.js进行编写，地图展示方面采用的是Cesium.js，旨在为湖北省内的试点水库提供信息，支持决策制定，以最大程度地减少洪水风险和提高公众安全。

2、部分代码未进行展示，目前项目正在优化中，后续再进行更新。

3、欢迎大家一起沟通交流。

## 项目启动

### 依赖安装

> npm install

### 启动项目

> npm run serve
